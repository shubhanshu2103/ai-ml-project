#  Clinical Appointment No-Show Predictor

##  Project Overview
This project is an AI-based healthcare operations system (Milestone 1) designed to predict patient appointment no-shows. By identifying high-risk patients, clinics can proactively intervene, prioritize resources, and reduce operational waste. 

##  Tech Stack
* **Frontend/UI:** Streamlit
* **Machine Learning:** Scikit-Learn (Decision Tree Classifier)
* **Data Processing:** Pandas, NumPy
* **Data Visualization:** Matplotlib

##  Local Setup Instructions
1. Clone this repository:
   `git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`
2. Navigate into the directory:
   `cd YOUR_REPO_NAME`
3. Install the required dependencies:
   `pip install -r requirements.txt`
4. Run the application:
   `streamlit run app.py`

##  Repository Structure
* `app.py`: The main Streamlit application script.
* `noshow_model.pkl`: The trained Decision Tree machine learning model.
* `requirements.txt`: List of Python dependencies.
* `README.md`: Project documentation.
